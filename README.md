# Webpack React ES6
Webpack React ES6 Boilerplate

# Prerequisites
Node v6.x
- Linux -> https://deb.nodesource.com/setup_6.x
- Other -> https://nodejs.org/en/download/

Yarn(optional)

# Supported features
HMR
TreeShaking
Flux with Redux
ES6

# Develop
yarn run start|npm run start

# Build
yarn run build|npm run start
